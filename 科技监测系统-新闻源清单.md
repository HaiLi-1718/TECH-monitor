## 补充说明

### 与 TECH_FEEDS 的关系

`LOCALTECH_FEEDS` 为精简后的本地科技监测配置：`biopharma`、`policy`、`security` 引用自完整的 `TECH_FEEDS` 定义；`tech`、`ai`、`startups` 使用独立列表以减少跨面板重复。

### INTEL_SOURCES（非 FEEDS 面板）

`INTEL_SOURCES`（情报动态 / intel 面板）在 `feeds.ts` 中单独定义，**不属于**上述 `FEEDS` 新闻面板，也不参与自动导出白名单。主要包括 Defense One、The War Zone、Foreign Policy、Krebs Security 等防务与情报类源。

### 容错机制（2026-06-28 更新）

- 客户端 proxy 层：`src/utils/proxy.ts` — 首屏无缓存请求带 15s 超时 + 1 次指数退避重试（800ms）
- Bootstrap 水合：`src/services/bootstrap.ts` — Web 端超时放宽至 4s / 7s（原 1.2s / 1.8s）
- 时间线面板：`src/components/TechEventsPanel.ts` — 重试间隔改为 2s → 5s → 10s 指数退避（原固定 15s）
- RSS 冷却策略：`src/services/rss.ts` — MAX_FAILURES 3 次（原 2 次）、冷却 2 分钟（原 5 分钟）
- freepatentsonline.com：加入 `RELAY_ONLY_DOMAINS`，始终通过 Railway 中继绕过 Vercel 封锁
