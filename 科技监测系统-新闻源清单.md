## 七、说明

1. **数据管线**：以上 5 个类别的 feed 由 `loadNews()`（`src/app/data-loader.ts`）并发抓取，优先走服务端预聚合 digest，失败时回退到逐 feed RSS 拉取。
2. **跨类别去重**：同一条新闻若在多个类别命中（如「InfoQ 中文」同时供安全/政策/科技话题），系统按 `techLikeCrossCategoryPriority`（`security > policy > biopharma > startups(已停用) > ai > tech`）顺序去重，确保同一故事只出现一次。
3. **已隐藏但保留代码的类别**：`startups`（创业与融资）feed 列表仍在 `src/config/feeds.ts` 中，仅被注释停用；如需恢复，取消注释并加回 `DEFAULT_PANELS`/`PANEL_CATEGORY_MAP` 即可，无需重新配置源。
4. **未使用的 scaffold**：`feeds.ts` 中还有一份更大的 `TECH_FEEDS`（含 vcblogs / regionalStartups / github / cloud / dev / podcasts 等更多分类），是另一变体的遗留配置，当前 `FEEDS`（=`LOCALTECH_FEEDS`）只挑选/复用了其中的 `biopharma`、`policy`、`security` 三块，其余未被加载，不计入本清单。
5. **不在此清单内**：地图图层数据源（会议、卫星、AIS 等结构化 API）、`INTEL_SOURCES`（防务/情报类 feed，仅用于源管理 UI，未接入本变体的实际抓取流程）。
